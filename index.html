Description

eesubmit makes it easy to submit your EE homework.

Easy Installation

Mac:

bash <(curl -s https://raw.githubusercontent.com/DakotaAndMichael/eesubmit/master/ei.bash)

Linux:

bash <(wget -O - https://raw.githubusercontent.com/DakotaAndMichael/eesubmit/master/ei.bash)

Uninstalling

eesubmit takes up very little space, but if you want to remove it, along with anything else it installed, add -u to the above scripts.

Mac:

bash <(curl -s https://raw.githubusercontent.com/DakotaAndMichael/eesubmit/master/ei.bash) -u

Linux:

bash <(wget -O - https://raw.githubusercontent.com/DakotaAndMichael/eesubmit/master/ei.bash) -u

Alternative Installation

  1. git clone https://github.com/DakotaAndMichael/eesubmit.git
  2. cd eesubmit/main
  3. ./macInstall.bash or ./linuxInstall.bash depending on your OS

Usage

  1. Change into your hw directory
  2. Name your scanned hw "hwHand<x>.pdf", where x is your homework number
  3. eesubmit
