Description

eesubmit makes it easier to submit your EE homework.

Installation

  1. Download the repository code
  2. If you're on Mac, run main/macInstall.bash. If you're on Linux, run main/linuxInstall.bash

Usage

  1. Move the pdf of your handwritten work into the same directory as your ipython notebook and name it "hwHand.pdf" (where is the hw assignment number)
  2. Execute the following command: eesubmit
